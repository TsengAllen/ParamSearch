# ParamSearch
## 前言
該程式主要透過四種參數選擇法（權重選擇法、標準差選擇法、島嶼面積選擇法、島嶼體積選擇法）來找尋屬於參數高原之參數，而各個參數選擇法中皆有以下幾個參數，分別是窗格比例、延伸圈數或標竿PR。本文主要是希望透過該程式找尋出每一參數選擇法對應不同商品以及交易策略時，其最適之窗格比例、延伸圈數或標竿PR為何。接下來我會講解基本的操作方法，至於程式背後的邏輯則需自行探討。
